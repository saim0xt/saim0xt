# PricePulse Icons

## Generating Icons

To generate the PNG icons for the extension:

1. Open `generate-icons.html` in a web browser
2. Download each icon size (16x16, 48x48, 128x128)
3. Save them in this directory as:
   - icon16.png
   - icon48.png
   - icon128.png

## Alternative: Use Online Tools

You can also use these online SVG to PNG converters:

1. Visit: https://cloudconvert.com/svg-to-png
2. Upload the `icon.svg` file
3. Convert to PNG at sizes: 16x16, 48x48, and 128x128
4. Download and rename files appropriately

## Icon Design

The PricePulse icon features:
- Purple gradient background (#667eea to #764ba2)
- White pulse wave line representing price tracking
- Price tag symbol
- Dollar sign indicating pricing focus
