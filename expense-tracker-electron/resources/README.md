# Resources Folder

This folder contains app resources like icons and images.

## Required Files

### For Windows Build
- `icon.ico` - Windows application icon (256x256 or higher)
- `icon.png` - Generic icon for development (512x512 recommended)

### Creating Icons

You can create icons using:
- [GIMP](https://www.gimp.org/) - Free image editor
- [Inkscape](https://inkscape.org/) - Free vector editor
- Online tools like [ICO Converter](https://www.icoconverter.com/)

### Icon Specifications

**Windows (.ico)**
- Size: 256x256 pixels minimum
- Format: ICO
- Recommended: Include multiple sizes (16, 32, 48, 128, 256)

**Development (.png)**
- Size: 512x512 pixels
- Format: PNG
- Transparent background recommended

## Placeholder Icons

For development, you can use emoji-based icons or download free icons from:
- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)
- [Noun Project](https://thenounproject.com/)

## Note

Icon files are not included in the repository. Add your own icons before building for production.
